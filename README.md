# Quarto Data Analysis Samples (R)

This repository contains two Quarto documents demonstrating my data wrangling, text processing, and visualization skills in R using **tidyverse** and other related packages. These examples were originally completed as part of *ESM 244: Advanced Data Analysis* and are intended to show my familiarity with Quarto workflows and reproducible data analysis.

---

## Contents

### 1. ramnath_qual_text_analysis.qmd
**Output:** `ramnath_qual_text_analysis.html`  

This notebook illustrates qualitative text wrangling and exploratory analysis using:

- `tidyverse`  
- `tidytext`  
- `pdftools`  
- `ggwordcloud`  
- `textdata`  

It includes text cleaning, tokenization, stop-word filtering, term frequency summaries, and visualizations. The .qmd includes the complete methods and code used for this assignment  

The HTML file is included because it renders cleanly and is small enough for GitHub.
